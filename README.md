# GT-IML

Development of a user interface prototype for an interactive machine learning system that supports Grounded Theory practitioners with building theories

### Prerequisites

- `node` >= 10.16
- `npm` >= 5.6

### Usage

```
cd frontend/
npm i
npm start
```
